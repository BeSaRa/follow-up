## Description

Text input and form field wrapper components for building forms. FormField provides a consistent layout with label, input slot, and error message.

## Deliverables

- [x]`UiInput` — styled text input with attribute selector `input[uiInput]`
- [x]`UiFormField` — wrapper with label, projected input, hint text, and error message
- [x]Support for disabled, readonly, and error states
- [x]Works with Angular Reactive Forms
- [x]Export from `@follow-up/ui`
- [x]Add to showcase page
